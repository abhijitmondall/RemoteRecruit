# RemoteRecruit

This repo contains the landing page for RemoteRecruit, built with React, TypeScript, and Vite.

Live demo: https://remoterecruti.netlify.app/

## Project setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Frameworks and libraries

- Vite
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- lucide-react
- react-icons

## Notes

- I implemented the page to match the Figma design as closely as possible using view-only access. Exact design system fidelity is limited without edit permissions.
- The footer social icons use `react-icons` to approximate the requested brand shapes; they are close matches, but not necessarily the exact original glyphs.
- A few spacing and color values are hard-coded in Tailwind because the full design system was not available. The layout can be made even more faithful by using the original design system colors and spacing directly.
